How the token works (read this once)

The token is embedded in the QR code URL — it is never written into the code file, so GitHub cannot detect or revoke it.

Step 1 — Make sure leaderboard.json exists in your repo with content: {"leaderboard":[]}

Step 2 — Paste your GitHub PAT in the field below.

Step 3 — Click Generate QR. The token is baked into the QR URL.

Step 4 — All participants scan this QR. Their phones load the token from the URL and scores sync live. ✓
Token is added to the QR URL as ?t=token — not stored anywhere in the file
