📷 QR Code & Hosting Setup

Scan to open the quiz on any phone — no app or Claude account needed
How to host this quiz for free (anyone can scan):

1. Go to github.com → sign in → create a new repository
2. Upload utkrishta_quiz.html and rename it to index.html
3. Go to Settings → Pages → Deploy from branch (main)
4. Your URL will be https://yourname.github.io/reponame/
5. Paste that URL above and click Generate for your QR code

For shared leaderboard across phones:
Open the hosted link on ALL phones → all scores save to the same browser storage on the hosted site. For multi-device sharing, each person should use the same hosted URL.
